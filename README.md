# Mono_Tutorial
本作品は制作途中のゲーム「Mono」のチュートリアル（Windows版）になります。
本リポジトリをダウンロードし、Mono.exeを起動すると遊ぶことができます。

### 概要
 - ジャンル：探索型2Dアクションゲーム
 - 制作人数（期間）：7人（2年～）
 - 内容説明：メトロイドヴァニアな2Dアクションゲームです。プレイヤーを操作したときの手触り感にこだわっており、アセットは全て自分たちで制作しています。(現在制作途中のゲームになります。)
 - 担当箇所：プレイヤー移動処理, テキスト処理, エフェクト処理
 - 使用技術：C#, Unity, React
